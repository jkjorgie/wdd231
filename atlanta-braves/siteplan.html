Atlanta Braves Site Plan

Site Name

"Atlanta Braves" because the website is all about the Atlanta Braves.

Site Purpose

The site provides a place for fans to learn about historic Atlanta Braves players and stay up to date on the latest team news.

Scenarios

  • Who are famous Atlanta Braves pitchers?
  • Who are famous Atlanta Braves third basemen?
  • How can I stay up to date on team news?

Color Schema

  • #13274F - primary color, text, borders, etc
  • #CE1141 - second primary color, header/footer, h tags, etc
  • #FFD700 - accent color, highlighting, special elements
  • #FAEBD7 - light background color
  • #A9A9A9 - darker background color

Typography

  • Open Sans - non h tag text
  • Montserrat - h tags
